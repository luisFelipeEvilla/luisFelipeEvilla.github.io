Business Intelligence (BI) and Data Analytics. By performing data analysis, valuable information can be obtained for decision-making, which can be used to improve business processes and manage the company. This is reflected in:

  • Increased productivity
  • Cost reduction
  • Resource optimization
  • Risk reduction

Various tools are used for this purpose, such as control panels with Google Data Studio to visualize and better understand the information, obtaining useful metrics and key insights for business process management. This is accompanied by the processing of the same data with programming tools such as Python, using libraries such as Numpy, Pandas, and Sklearn.
